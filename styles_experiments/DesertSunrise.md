# DesertSunrise

## Brand Direction
DesertSunrise should feel optimistic, warm, and expansive. Think early light in open landscapes: soft contrast, earthy tones, and energetic CTAs.

## Visual System
- Core palette:
  - Sand and clay neutrals for base surfaces.
  - Sunrise gradients (peach, amber, rust) for emphasis.
  - Deep dusk text color for readability.
- Surfaces:
  - Gentle elevation, large rounded corners, low-noise backgrounds.
- Typography:
  - Bold section headings + clear body text with generous line-height.

## Layout System
- Keep page rhythm breathable with larger vertical spacing.
- Use section dividers via tonal shifts, not heavy borders.
- Keep key CTA visible above the fold on mobile.

## Components
- Primary buttons: warm gradient fill.
- Secondary buttons: sand background + subtle border.
- Cards: layered neutrals with soft shadows.
- Tags: color-coded by feature context.

## Feature Mapping
- Onboarding: welcoming, bright, low anxiety.
- Friends/Dating Discovery: high-contrast card actions and clear outcomes.
- Matches & Chat: warm accents for active conversation and unread states.
- Campfire: strongest thematic fit; event discovery feels energetic and communal.
- Profiles: storytelling-first with photos, prompts, and travel timeline.

## Interaction
- Use warm hover/focus states consistently.
- Keep motion smooth and subtle (180-280ms).
- Use one animation pattern per context to avoid noise.

## Implementation Plan
1. Define color tokens and gradient tokens.
2. Update shared primitives (`app-card`, buttons, inputs).
3. Apply to navigation, shell, and major pages.
4. Audit contrast/accessibility and refine edge states.

## Quality Checklist
- Warm identity is consistent across all tabs.
- CTAs are instantly distinguishable from passive elements.
- Text readability remains strong on all backgrounds.
